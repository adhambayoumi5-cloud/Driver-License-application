# Driver License Application

A C#/.NET project for managing driver license applications.  
This application handles data access, business logic,windows form,ado.net,sqlserver and user interface for processing applications efficiently.

- Manage driver license applications
- Store applicant data securely
- Handle business logic validations
- Connect to database using a clean Data Access layer
